# SimpleXmlDoc

Simple XML Document in JavaScript, WIP

## Installation

Via npm on Node:

```
npm install simplexmldoc
```

You can include `lib/simplexmldoc.js` directly into your HTML page. It declares the `sl` namespace.

## Usage

Reference in your Node.js program:
```javascript
var sl = require('simplexmldoc');
```
